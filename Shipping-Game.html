Shipping Game

Previous Next

This project was the final project for my Object Oriented Programming class. The framework of the game was provided, and the students' jobs were to implement an algorithm to direct trucks to certain cities in order to pick up and then deliver parcels. We were graded based on our algorithms' scores in the game which were based off of the amount of time it took to deliver all the parcels. In order to implement the algorithm, I had to create a Min Heap and use that as part of an implementation of Dijksta's algorithm of finding shortest paths.
