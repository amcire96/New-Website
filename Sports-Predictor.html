Sports Predictor

Previous Next

After taking a supervised machine learning course, I wanted to apply my machine learning knowledge to solve something that interested me. At that time, fantasy basketball had been started recently, so I wondered if I could try to predict players' stats given their previous game statistics and given game information for their next game. The program eventually transformed into a daily fantasy sports predictor that I used for FanDuel (I actually put in money into this)!

I scraped the statistics and game information off of ESPN using Python's lxml library and then used scikit-learn's models to fit and predict. In the future, I want to improve my data storage, as right now I am naively writing all of the scraped data into local files and also create a user interface, either as an Android app or a web app.

You can find the code here: https://github.com/amcire96/Sports-Predictor.
