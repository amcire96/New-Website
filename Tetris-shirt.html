Tetris T-shirt

Previous Next

The Tetris T-shirt was a follow-up on the Tetris game that I programmed into the computer. The idea for the shirt came when I found a video of a tie that had a 3x10 LED display attached to a tie that illuminated the game of tetris. However, Tetris cannot realistically be played on a 3x10 board, so I wanted to see if I could create the project on a bigger scale with a larger display. I sewed a 10 by 20 LED grid onto a t-shirt as a board. Then, I found the Tetris tie's arduino code and implemented my algorithm in place of the current algorithm, which was rather terrible at playing tetris. After uploading the code onto the Arduino Mini and plugging both the Arduino Mini and the LED's into a portable battery, the project was complete!
