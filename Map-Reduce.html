Map-Reduce

Previous Next

This project was the final project of my functional programming class. In order to implement Map-Reduce, we created a controller and workers. The workers had to perform the map and the reduce functions, while the controller was responsible for distributing tasks for the workers. We used our implemented Map-Reduce to perform 3 different tasks: count words, find common friends in a network, and find the prime factors of the product of a list of numbers. For each application, we had to decide how to split up the data for the workers; for example in the word counting application, the controller split the file into lines, and each worker received a line to count words from.
