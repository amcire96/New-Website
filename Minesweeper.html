Minesweeper

Previous Next

I programmed this game as the final course project for my introductory Java class. As a pretty experienced Minesweeper player, it was really interesting and fun to build a game that I knew so well. Programming to actual game into Java was not terribly difficult, but the graphic portion of the project presented a bit more of a challenge. After a bit of fiddling, I am proud of the final project which, along with the game board, includes a menu bar, a couple labels regarding the timer and number of bombs left, a restart button, and a couple menus before the game starts regarding the game settings. The game supports the normal Minesweeper features of placing flags and double clicking to open all neighbor locations. It even has sound effects!
