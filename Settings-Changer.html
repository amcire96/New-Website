Settings Changer App

Previous Next

After learning Android Programming, I created this app which allows the user to change certain settings at a specified time on specified days of the week. The motivation for this project came while I was interning. At work, I would turn on Wifi, disable Bluetooth so that my smart-watch would not vibrate during work, and turn the volume of the phone off. Then after work, when going outside, I would turn off Wifi, turn on Bluetooth, and turn my volume up. This was a hassle, so I decided to create this app which allows the user to turn on/off Wifi and Bluetooth and set the volume level at a specified time on a specified day of the week.

You can find the code here: https://github.com/amcire96/SettingsChanger.
